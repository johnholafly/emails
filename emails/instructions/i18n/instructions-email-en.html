Activate your data plan at your destination

For iOS devices

  • Go to Settings
  • → Cellular

Select Default Voice Line and set Holafly as your preferred line to make and receive calls.

Return to Cellular and select the Holafly eSIM, turn on this line and activate data roaming. Also, make sure to select your new plan as primary for mobile data.

For Android devices

Samsung

  • Go to Settings → 
  • Connections → 
  • SIM manager

Tap on Mobile data and select Holafly as your preferred line for data. Then tap on Calls and select Holafly as your preferred line to make and receive calls.

Return to Connections, select Mobile networks and turn on Data roaming

Google pixel

  • Go to Settings → 
  • Network & Internet

Tap on Calls & SMS and select Holafly as your preferred line to make and receive calls.

Return to Network & Internet, select SIMs and tap on the Holafly eSIM, turn on Use SIM. A message will come out asking if you want to switch to Holafly for mobile data, confirm it and wait. Then turn on Roaming.

Activate your data plan at your destination

  • Go to Settings
  • → Cellular

Select the Holafly eSIM, turn on this line and activate data roaming. Also, make sure to select your new plan as primary for mobile data.

What you need to know

Install the day you travel

We recommend installing the eSIM the day you travel, a couple of hours before your trip.
 

Scan from the Settings menu

Keep in mind some phones won’t recognize the code if you scan it directly from the Camera app.

Don't delete the eSIM

You can only scan the code once. If you remove the eSIM, you can't recover it.
 

Turn on Data Roaming

Activate Data Roaming in Settings for the eSIM to work.
